# Valentine's Day Webpage

This project is a romantic-themed webpage designed to celebrate Valentine's Day. It features a central message asking "Elise, Would you be my Valentine?", along with interactive buttons that trigger animations based on user interactions.

## Project Structure

```
valentines-day-webpage
├── index.html       # Main content of the webpage
├── styles.css       # Romantic styling and animations
├── script.js        # Button behavior and animations
└── README.md        # Project documentation
```

## Files Description

- **index.html**: Contains the main content of the webpage, including the central message and buttons for user interaction. The structure is designed to be engaging and romantic.

- **styles.css**: Provides the styling for the webpage, utilizing a color palette of pink, red, and white. It includes styles for the background, buttons, text, and animations for hearts and messages.

- **script.js**: Manages the behavior of the buttons. It includes functions to trigger heart explosion animations when "Yes" is clicked and to display a refusal message with an animation when "No" is clicked. The refusal message prompts the user until they choose "Yes".

## Setup Instructions

1. Clone the repository or download the project files.
2. Open `index.html` in a web browser to view the webpage.
3. Customize the content and styles as desired by editing the respective files.

## Features

- Interactive buttons that respond to user clicks.
- Animated hearts that explode when "Yes" is clicked.
- A refusal message that prompts the user until they choose "Yes".
- Romantic design suitable for Valentine's Day celebrations.
